================================================================================
START HERE - Food Redistribution App
================================================================================

1. Open this project in VS Code or Android Studio (open the folder
   "food_redistribution_app", not the parent folder).

2. Install dependencies (run in terminal from this app folder):
      flutter pub get

3. Run the app:
      flutter run
   Or use the Run/Debug button in your IDE.

4. Where is the code?
   - Read docs/STRUCTURE.txt for a simple map of all folders.
   - Your app code is under:  lib/
   - Screens by role:  lib/screens/  (auth, donor, ngo, volunteer, admin)
   - Services and data:  lib/services/  and  lib/models/

5. Scripts (optional):
   - scripts/run_tests.bat  or  scripts/run_tests.sh  - run tests
   - scripts/test_matching.dart - matching algorithm test (dart run scripts/test_matching.dart)
   - scripts/verify_schemas.js - schema check (Node.js)

6. One-time git cleanup (if your team had merge conflicts before):
   - See docs/CLEAN_GIT_ONCE.txt
